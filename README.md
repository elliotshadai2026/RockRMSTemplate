# RockRMS Template Repository

Custom themes and templates for RockRMS church management system.

## Live Demo

Visit the live preview: [https://elliotshadai2026.github.io/RockRMSTemplate/](https://elliotshadai2026.github.io/RockRMSTemplate/)

## BBCC RockRMS Theme

A mobile-first theme with blue navigation and BBCC branding for RockRMS.

### Features

- **Blue Navigation Background** - #0d6efd (BBCC Blue)
- **Spark Theme Foundation** - Built on Spark theme as starting point
- **RockRMS Spec Compliant** - Bootstrap 3.4.1, Font Awesome 4.7.0
- **Mobile First Design** - Responsive layout with hamburger menu
- **BBCC Branding** - Church colors and logo integration

### Preview Links

- [BBCC Theme Preview](bbcc-rockrms-theme.html)
- [Repository Landing Page](index.html)

### Color Scheme

- **Primary Blue**: `#0d6efd` (Navigation, headers)
- **Accent Orange**: `#ee7625` (Buttons, active states)
- **Background**: `#f8f9fa` (Light gray)
- **Text**: `#343a40` (Dark gray)

### Mobile Features

- Hamburger menu for mobile navigation
- Responsive sidebar with smooth animations
- Touch-friendly buttons and controls
- Optimized for tablets and phones

## Usage

1. Clone the repository:
   ```bash
   git clone https://github.com/elliotshadai2026/RockRMSTemplate.git
   ```

2. Open `bbcc-rockrms-theme.html` in your browser to preview the theme.

3. For RockRMS integration, extract the CSS/JS and integrate into your RockRMS theme.

## Development

This theme is built with:
- Bootstrap 3.4.1 (RockRMS standard)
- Font Awesome 4.7.0
- jQuery 3.7.1
- Custom CSS with CSS variables
